# PRODUCT

## Product Position

AI Studio is a desktop application for AI-assisted software development.

The user should feel like they are managing a team of AI engineers rather than talking to a chatbot.

---

# Primary Users

Initially:

Single developer using Hermes.

Future:

Individuals.

Teams.

Researchers.

AI-assisted software companies.

---

# Core Experience

The product revolves around six major workspaces.

## Workspace

Browse project files.

Open folders.

Track modifications.

---

## Chat

Communicate with AI agents.

Markdown support.

Code rendering.

Streaming response.

History.

---

## Console

Display live execution.

Tool calls.

Logs.

Progress.

Errors.

---

## Tasks

Current work.

Pending work.

Future work.

---

## Agents

Manage available AI agents.

Switch between providers.

Future Multi-Agent support.

---

## Settings

Workspace configuration.

Themes.

Model configuration.

Paths.

Plugins.

---

# Design Philosophy

The interface should remain calm.

No unnecessary animations.

No clutter.

No information overload.

Professional development environment.
