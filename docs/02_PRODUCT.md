# PRODUCT

## Product Position

AI Studio is a desktop application for AI-assisted software development.

The user should feel like they are managing a team of AI engineers rather than talking to a chatbot.

---

# Primary Users

Initially:

Single developer using Hermes.

Future:

Individuals.

Teams.

Researchers.

AI-assisted software companies.

---

# Core Experience

The product revolves around several major workspaces.

## Workspace

Browse project files.

Open folders.

Track modifications.

File operations (create, rename, delete, copy, move).

Preview files.

Open in Editor.

---

## Chat

Communicate with AI agents.

Markdown support.

Code rendering.

Streaming response.

History.

---

## Dashboard (v0.2.0+)

Project health overview.

Milestone progress.

Development Intelligence.

Current task + next action.

---

## Editor (v0.3.0+)

Edit project files.

Diff comparison.

Patch application.

Single write gate with audit trail.

---

## Console

Display live execution.

Tool calls.

Logs.

Progress.

Errors.

---

## Command Palette (v0.3.0+)

Quick command execution (Ctrl+P / Ctrl+K).

Search across all registered commands.

Keyboard shortcut discovery.

---

# Design Philosophy

The interface should remain calm.

No unnecessary animations.

No clutter.

No information overload.

Professional development environment.
