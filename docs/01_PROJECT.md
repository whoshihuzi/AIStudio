# PROJECT

## Project Name

AI Studio

---

# Project Definition

AI Studio is an AI Native IDE.

It is not:

* A chat application
* A Hermes GUI
* A code editor
* A terminal replacement

Although it contains all of those capabilities.

---

# Mission

Create a software platform where AI agents can participate in software engineering as first-class collaborators.

---

# Vision

Future software development should become:

Human

↓

AI Planning

↓

AI Coding

↓

Human Review

↓

AI Testing

↓

AI Deployment

AI Studio exists to make this workflow possible.

---

# Core Values

## Human First

Humans always own the project.

AI never makes product decisions.

---

## Agent Agnostic

Every AI agent is replaceable.

The application must never depend on a specific model.

Hermes is only the first implementation.

---

## Long-term Evolution

Every architecture decision must support future expansion.

Avoid temporary solutions.

Avoid hardcoded assumptions.

---

## Incremental Development

Always ship working software.

Never build everything before testing.

---

## Maintainability

Readable architecture is more important than clever implementation.

---

# Success Criteria

Phase 1

Hermes can be used entirely through AI Studio.

No PowerShell required.

Phase 2

Project management.

Phase 3

Multi-Agent.

Phase 4

Workflow Automation.

Phase 5

AI Studio develops AI Studio.

---

# Non Goals

AI Studio is NOT trying to compete with VS Code as a traditional editor.

AI Studio is creating a new workflow centered around AI collaboration.

This distinction must always remain clear.
